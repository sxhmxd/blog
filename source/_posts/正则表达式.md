### 匹配小括号里面的内容

```
  let reg =  /[^(][a-zA-Z0-9]+(?=\))/g
  let str = "(9001)(9002)(9003)"
  str.match(reg)
   // ['9001','9002','9003']
```
### 将中括号替换为点

```
  let str = 'a[12][334]'
  // 方法一
    str.replace(/\[/g, ".").replace(/\]/g, "");
      // a.12.334
  // 方法二
    str.replace(/\[([^\[\]]*)\]/g, ".$1");
      // a.12.334
```
