###  图例特殊化定义

```javaScript
  //  实现 图例下面显示横线
      legend: {
          show: true,
          icon: this.icon,
          top: "center",
          left: "60%",
          data: arrName,
          width: 100,
          padding: [0, 0],
          itemGap: 0,
          formatter: function (name) {
            return `{dd|}{title|${name}}{percent|${objData[name].percent}}\n\n{hr|}`;
          },
          textStyle: {
            padding: [0, 0, 0, -30],
            rich: {
              dd: {
                backgroundColor: {
                  // image: this.image,
                },
              },
              title: {
                fontSize: 14,
                lineHeight: 20,
                color: "#fff",
                padding: [15, 50, 0, 30],
              },
              hr: {
                borderColor: "#707377",
                width: 180,
                borderWidth: 0.5,
                height: 0,
              },
              percent: {
                fontSize: 14,
                lineHeight: 20,
                color: "#fff",
                padding: [15, 0, 0, 0],
              },
            },
          },
        },
```
